Create or update test environment / Creare Ambiente di Test

\n

\nThis module creates an test environment with test data.\n

\nThe behaviour of this module, depends heavily from following PYPI modules:\n

\n

    \n
  • z0bug_odoo>=1.0.5.6: contains records data
    • \n
  • clodoo>=0.3.53.1: contains the odoo version depending translation
    • \n
  • odoo_score==1.0.3: odoo supercore
    • \n
